1.3 Laplace Transform Revision#

Recall from DEM that the Laplace transform (LT) of a function is defined as follows.


Definition 1.3.1 The Laplace transform of the function, \(f(t)\), is given by

(8)#\[ \mathcal{L}[f] = \tilde{f}(t) = \int_0^\infty e^{-st}f(t)\mathrm{d}t. \]

Example 1.3.1 Here we use (8) directly to determine the Laplace transforms of some simple functions.

  1. For \(\displaystyle f(t)=1, \mathcal{L}[1] = \int_0^\infty e^{-st}\mathrm{d}t = \left[-\frac{1}{s}e^{-st}\right]_0^\infty = \frac{1}{s}\).

  2. For \(\displaystyle f(t)=t, \mathcal{L}[t] = \int_0^\infty te^{-st}\mathrm{d}t = \left[-\frac{1}{s}e^{-st}t\right]_0^\infty+\int_0^\infty \frac{1}{s}e^{-st}\mathrm{d}t = \frac{1}{s^2}\).

  3. For \(\displaystyle f(t)=\frac{\mathrm{d}y}{\mathrm{d}t}, \mathcal{L}\left[\frac{\mathrm{d}y}{\mathrm{d}t}\right] = \int_0^\infty e^{-st}\frac{\mathrm{d}y}{\mathrm{d}t}\mathrm{d}t = \left[e^{-st}y\right]_0^\infty+\int_0^\infty se^{-st}y\mathrm{d}t = \frac{1}{s} = -y(0)+s\tilde{y}(s)\).

  4. For \(\displaystyle f(t)=e^{at}, a\) constant,

\[ \quad\mathcal{L}[e^{at}] = \int_0^\infty e^{-st}e^{at}\mathrm{d}t = \int_0^\infty e^{-(s-a)t}\mathrm{d}t = \left[-\frac{1}{s-a}e^{-(s-a)t}\right]_0^\infty = \frac{1}{s-a}, ~s>a. \]

The following theorem is particularly useful when computing LTs.


Theorem 1.3.1 (Linearity) The Laplace transform is a linear operation; that is, for any functions \(f(t)\) and \(g(t)\) whose transforms exist and any constants \(a, b\in\mathbb{R}\) the transform \(af(t)+bg(t)\) exists and

\[ \mathcal{L}[af+bg] = a\mathcal{L}[f] + b\mathcal{L}[g]. \]

Example 1.3.2 Using the result \(\displaystyle\mathcal{L}[e^{at}] = 1/(s-a)\) and the fact that the Laplace transform is linear, determine \(\displaystyle \mathcal{L}[\cos(at)]\) and \(\displaystyle \mathcal{L}[\sin(at)]\).

Solution

Using DeMoivre’s theorem we know that

\[ e^{iat} = \cos(at)+i\sin(at). \]

Also, we have that

\[ \mathcal{L}[e^{iat}] = \frac{1}{s-ia} = \frac{s}{s^2+a^2}+\frac{ia}{s^2+a^2}. \]

Hence, equating the real and imaginary parts and using linearity gives

\[ \mathcal{L}[\cos{at}] = \frac{s}{s^2+a^2} \]

and

\[ \mathcal{L}[\sin{at}] = \frac{a}{s^2+a^2}. \]

Another useful result is the so-called shifting theorem.


Theorem 1.3.2 If \(f(t)\) has the transform \(\tilde{f}(s)\), then \(\displaystyle e^{at}f(t)\) has transform \(\displaystyle \tilde{f}(s-a)\) (where \(s-a>0\)). In formulas

\[ \mathcal{L}[e^{at}f(t)] = \tilde{f}(s-a) \]

and

\[ e^{at}f(t) = \mathcal{L}^{-1}[\tilde{f}(s-a)]. \]

Example 1.3.3 Use the shifting theorem to show that \(\displaystyle \mathcal{L}[e^{at}\cos{\omega t}] = \frac{s-a}{(s-a)^2+\omega^2}\).

Solution

Denoting by \(f(t) = \cos{\omega t}\) we have by the shifting theroem that

\[ \mathcal{L}[e^{at}\cos{\omega t}] = \tilde{f}(s-a) \]

where \(\displaystyle \tilde{f}(s) = \frac{s}{s^2+\omega^2}\), so that

\[ \mathcal{L}[e^{at}\cos{\omega t}] = \frac{s-a}{(s-a)^2+\omega^2} \]

as required.

Finding inverse transforms using partial fractions#

The inverse Laplace transform is written

\[ \mathcal{L}^{-1}[\tilde{f}] = f. \]

A common situation is when \(\displaystyle \tilde{f}(s)\) is a polynomial, or more generally, a rational function (i.e. a ratio of polynomials), then we can often use partial fractions to simplify the expression.


Example 1.3.5 Show that \(\displaystyle\mathcal{L}^{-1}\left[\frac{1}{2s+3}\right] = \frac{1}{2}e^{-3t/2}\).

Solution

By linearity we can write

\[ \mathcal{L}^{-1}\left[\frac{1}{2s+3}\right] = \frac{1}{2}\mathcal{L}^{-1}\left[\frac{1}{s+3/2}\right] = \frac{1}{2}e^{-3t/2}. \]

Here, we have used the result in the third row of Table 1.


Example 1.3.6 Show that \(\displaystyle \mathcal{L}^{-1}\left[\frac{1+s}{(s+3)(s-2)}\right] = \frac{2}{5}e^{-3t}+\frac{3}{5}e^{2t} \)

Solution

Using partial fractions we can rewrite

\[ \frac{1+s}{(s+3)(s-2)} = \frac{2}{5}\frac{1}{s+3} + \frac{3}{5}\frac{1}{s-2} \]

so that

\[\begin{split} \mathcal{L}^{-1}\left[\frac{1+s}{(s+3)(s-2)}\right] &= \frac{2}{5}\mathcal{L}^{-1}\left[\frac{1}{s+3}\right] + \frac{3}{5}\mathcal{L}^{-1}\left[\frac{1}{s-2}\right]\\ &= \frac{2}{5}e^{-3t} + \frac{3}{5}e^{2t}. \end{split}\]
