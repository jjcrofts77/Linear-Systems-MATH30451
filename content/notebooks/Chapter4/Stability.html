4.2 Linear Stability#

Having discussed the controllability and observability of certain types of linear systems, we shall now consider the question of stability and we begin with homogeneous, or free, systems of the form

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d} t} = A\mathbf{x} \qquad : t\geq 0. \]

Such a system is said to have an equilibrium point \(\mathbf{x}_e\) if

\[ A\mathbf{x}_e = 0, \text{ for } t\geq 0. \]

These points are so called because if we take the condition \(\displaystyle\mathbf{x}(0)=\mathbf{x}_e\) the solution of the given system is \(\displaystyle\mathbf{x}(t)=\mathbf{x}_e\) for all \(t\geq 0\).

The most obvious equilibrium point of all homogeneous systems is \(\mathbf{0}\) and, in fact, this is the only one we really need to consider since any other equilibrium point \(\mathbf{x}_0\) can be translated to \(\mathbf{0}\) by a simple change of variables as follows. If we take \(\displaystyle\mathbf{x}=\mathbf{x}_e+\mathbf{x}'\), we see that

Lyapunov Stability

\[\begin{split} \frac{\mathrm{d}\mathbf{x}'}{\mathrm{d}t} &= \frac{\mathrm{d}}{\mathrm{d}t}\left(\mathbf{x}-\mathbf{x}_e\right) = \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x} =A(\mathbf{x}_e+\mathbf{x}'),\\ &=A\mathbf{x}_e+A\mathbf{x}' = A\mathbf{x}', \end{split}\]

i.e., \(\displaystyle\mathbf{x}'\) satisfies the same system of differential equations as \(\mathbf{x}\) and when \(t=0\) \(\displaystyle\mathbf{x}'=\mathbf{x}_e-\mathbf{x}_e=0\).

Now, a system of the above form is said to be stable (or Lyapunov stable) if, given \(\epsilon>0\), there is a \(\delta>0\) such that

\[ ||\mathbf{x}(t)||<\epsilon \text{ for all } t\geq 0, \]

provided \(\displaystyle ||\mathbf{x_0}||<\delta\).

Asymptotic Stability

In effect we are saying that if an acceptable limit for the displacement of the state from the equilibrium point is set, this limit will not be exceeded provided the original disturbance from the equilibrium point is within another limit, which can be found. A much stronger form of stability occurs when the system is asymptotically stable, in which case \(\mathrm{d}\mathbf{x}(t)\to \mathbf{0}\) as \(t\to\infty\) whatever the original displacement.


Example 4.2.1 Let

\[\begin{split} A=\begin{pmatrix}0&1\\-1&0\end{pmatrix}. \end{split}\]

If

\[ \mathbf{x}_e=\begin{pmatrix}\alpha, & \beta\end{pmatrix}^T, \]

is an equilibrium point for the system then

\[\begin{split} A\mathbf{x}_e = \begin{pmatrix}0&1\\-1&0\end{pmatrix}\begin{pmatrix}\alpha\\\beta\end{pmatrix} = \begin{pmatrix}\beta\\-\alpha\end{pmatrix} = \begin{pmatrix}0\\0\end{pmatrix}. \end{split}\]

Clearly, the only such equilibrium point in this case is the origin.

Now, it can be shown that

\[\begin{split} e^{At} = \begin{pmatrix}\cos{t}&\sin{t}\\-\sin{t}&\cos{t}\end{pmatrix}. \end{split}\]

Therefore, if

\[\begin{split} \mathbf{x}(0)=\mathbf{x}_0=\begin{pmatrix} x_{10}\\x_{20}\end{pmatrix}, \end{split}\]

the solution of the system is

\[\begin{split} \mathbf{x}(t) = e^{At}\mathbf{x}_0 = \begin{pmatrix}\cos{t}&\sin{t}\\-\sin{t}&\cos{t}\end{pmatrix}\begin{pmatrix} x_{10}\\x_{20}\end{pmatrix} =\begin{pmatrix}\cos{(t)}x_{10}+\sin{(t)}x_{20}\\-\sin{(t)}x_{10}+\cos{(t)}x_{20}\end{pmatrix}, \end{split}\]

and

Here we use the Euclidean norm to measure the magnitude of the vector

\[\mathbf{u} = [u_1, u_2,\ldots, u_n]^T\in\mathbb{R}^n\]

which is defined as

\[ ||\mathbf{u}|| = \sqrt{u_1^2+u_2^2+\cdots+u_n^2}. \]
\[\begin{split} ||\mathbf{x}(t)||^2 &= (\cos{(t)}x_{10}+\sin{(t)}x_{20})^2+(-\sin{(t)}x_{10}+\cos{(t)}x_{20})^2,\\ &< (x_{10}+x_{20})^2+(x_{10}+x_{20})^2 = 2(x_{10}+x_{20})^2,\\ &\leq 2||2\mathbf{x}_0||^2 = 8||\mathbf{x}_0||^2<\epsilon^2, \end{split}\]

if \(\displaystyle ||\mathbf{x}_0||<\frac{\epsilon}{\sqrt{8}}\). This system is therefore stable but clearly not asymptotically stable.


Below we use python to plot the solution from Example 4.2.1 for \(t\in[0,6\pi]\) starting from the initial condition \(\mathbf{x}_0 = [1,1]^T\). It should clear that the solution is bounded for all time (i.e. Lyapunov stable).

# load packages for basic maths and plotting solutions
import numpy as np # matlab-like commands 
import matplotlib.pyplot as plt # plotting tools

ax.plot3D(t,np.cos(t)+np.sin(t),-np.sin(t)+np.cos(t))

# add on initial and final points
ax.scatter(0,1,1)
ax.scatter(t[-1],np.cos(t[-1])+np.sin(t[-1]),-np.sin(t[-1])+np.cos(t[-1]))

Example 4.2.2 If

\[\begin{split} A' = \begin{pmatrix}-1&1\\-1&-1\end{pmatrix} = A-I_2, \end{split}\]

then

\[\begin{split} e^{A't} = e^{At}e^{-t}= e^{-t}\begin{pmatrix}\cos{t}&\sin{t}\\-\sin{t}&\cos{t}\end{pmatrix}. \end{split}\]

In this case

\[\begin{split} \mathbf{x}(t) = e^{-t}\begin{pmatrix}\cos{(t)}x_{10}+\sin{(t)}x_{20}\\-\sin{(t)}x_{10}+\cos{(t)}x_{20}\end{pmatrix}, \end{split}\]

which tends to zero as \(t\) tends to infinity. The corresponding system is therefore asymptotically stable.


Below we use python to plot the solution from Example 4.2.2 for \(t\in[0,6\pi]\) starting from the initial condition \(\mathbf{x}_0 = [1,1]^T\). It should clear that the solution is asymptotically stable since it approaches the origin for increasing \(t\).

Considerations of stability for systems of the above type may be linked to the eigenvalues of the matrix \(A\) using the fact that the solution of such a system is given by

\[ \mathbf{x}(t) = e^{At}\mathbf{x}_0 = He^{Jt}H^{-1}\mathbf{x}_0. \]

For the Jordan canonical form

\[\begin{split} J = \begin{pmatrix} J_{n_1}&0&\ldots&0\\0&J_{n_2}&\ldots&0\\\vdots&\vdots&\ddots&\vdots\\0&0&\ldots&J_{n_k}\end{pmatrix}, \end{split}\]

we know that

\[\begin{split} e^{Jt} = \begin{pmatrix} e^{J_{n_1}t}&0&\ldots&0\\0&e^{J_{n_2}t}&\ldots&0\\\vdots&\vdots&\ddots&\vdots\\0&0&\ldots&e^{J_{n_k}t}\end{pmatrix}, \end{split}\]

and \(\displaystyle e^{J_{n_i}t}\) is either of the form \(\displaystyle e^{\lambda_i}I_{n_i}\) or

(18)#\[\begin{split} e^{J_{n_i}t} = e^{\lambda_it}\begin{pmatrix}1&t&\frac{t^2}{2!}&\ldots&\frac{t^{n_i-1}}{(n_i-1)!}\\ 0&1&t&\ldots&\frac{t^{n_i-2}}{(n_i-2)!}\\\vdots&\vdots&\vdots&&\vdots\\ 0&0&0&\ldots&1\end{pmatrix}. \end{split}\]

It follows that

\[ \mathbf{x}(t) = \sum_{i=1}^k\mathbf{p}_i(t)e^{\lambda_it}, \]

where \(\displaystyle\mathbf{p}_i(t)\) in each case is a column \(n\)-vector whose elements are polynomials in \(t\).

The following conclusions may then be arrived at:

  1. if any of the eigenvalues of \(A\) has a positive real part, the system cannot be stable;

  2. if all of the eigenvalues of \(A\) have negative real parts, the system is asymptotically stable;

  3. If all of the eigenvalues of \(A\) have non-positive real part and any eigenvalue which has a zero real part is not repeated (ruling out the possibility of a block like (18) in \(\displaystyle e^{At}\)), the system is stable.

Note

When eigenvalues with zero real parts are repeated, knowledge of the corresponding Jordan canonical form is required to decide whether the corresponding homogeneous system is stable.


Routh-Hurwitz Stability Criteria#

One of the most useful tests, which can be carried out on the characteristic polynomial

\[ (-1)^{n}\left(\lambda^n+a_1\lambda^{n-1}+\cdots+a_{n-1}\lambda+a_n\right), \]

to determine whether all the eigenvalues have negative real parts (so that the system \(\displaystyle\frac{\mathrm{d}\mathbf{x}}{\mathrm{d} t}=A\mathbf{x}\) is asymptotically stable), requires the formulation of the \(n\times n\) Hurwitz (or Routh-Hurwitz) matrix

\[\begin{split} H = \begin{pmatrix} a_1&a_3&a_5&\ldots&a_{2n-1}\\1&a_2&a_4&\ldots&a_{2n-2}\\0&a_1&a_3&\ldots&a_{2n-3}\\ \vdots&\vdots&\vdots&&\vdots\\0&0&0&\ldots&a_n\end{pmatrix} \qquad \text{(where $a_r=0$ if $r>n$)}. \end{split}\]

Then, all eigenvalues of \(A\) have negative real parts if and only if all the leading principal minors of \(A\),

\[\begin{split} \text{i.e., } H_1=a_1, H_2 = \left|\begin{matrix}a_1&a_3\\1&a_2\end{matrix}\right|, H_3 = \left|\begin{matrix}a_1&a_3&a_5\\1&a_2&a_4\\0&a_1&a_3\end{matrix}\right|,\ldots H_n=a_nH_{n-1}, \end{split}\]

are positive.


Example 4.2.3 If a matrix has characteristic polynomial

\[ \lambda^4+8\lambda^3+19\lambda^2+33\lambda+36, \]

then

\[\begin{split} H=\begin{pmatrix}8&33&0&0\\1&19&36&0\\0&8&33&0\\0&1&19&36\end{pmatrix}. \end{split}\]

Here

\[\begin{split} H_1=8,\quad H_2=\left|\begin{matrix}8&33\\1&19\end{matrix}\right|=119,\quad H_3=\left|\begin{matrix}8&33&0\\1&19&36\\0&8&33\end{matrix}\right|=1623 \text{ and } H_4=36H_3>0, \end{split}\]

so that the corresponding system can be seen to be asymptotically stable. In fact, the characteristic polynomial above factorises to give

\[ (\lambda+3)(\lambda+4)(\lambda^2+\lambda+3), \]

which leads to the eigenvalues \(\displaystyle\lambda=-3, -4\) and \(\displaystyle\frac{-1\pm\sqrt{11}i}{2}\).


Note

  1. If any of the coefficients in the characteristic polynomial are non-positive, the given system is not asymptotically stable.

  2. In the case of a quadratic of the form \(\displaystyle\lambda^2+a\lambda+b\), the resulting eigenvalues have negative real parts if and only if \(a>0\) and \(b>0\).
