4.1 Controllability and Observability#

Controllability#

In the last chapter we obtained a formula for the solution of a linear system of first order differential equations of the form

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d} t} = A\mathbf{x} + \mathbf{f}(t)\quad : \quad \mathbf{x}(0) = \mathbf{x}_0, \]

i.e.,

\[ \mathbf{x}(t) = e^{At}\mathbf{x}_0+\int_0^te^{A(t-\tau)}\mathbf{f}(\tau)\mathrm{d}\tau. \]

Suppose that we now rewrite the ‘forcing effect’ in a form that shows explicitly how many ‘control functions’ can be used to try to influence the behaviour of the system,

\[ \text{i.e., } \mathbf{f}(t) = B\mathbf{u}(t), \]

where \(B\) is a constant \(n\times p\) matrix and \(\mathbf{u}(t)\) is a column \(p\)-vector consisting of these ‘control functions’.

In these circumstances the solution of a system of the above form at time \(t=t_f\) is given by

(17)#\[\begin{split} \mathbf{x}(t_f) &= e^{At_f}\mathbf{x}_0+\int_0^{t_f}e^{A(t_f-\tau)}B\mathbf{u}(\tau)\mathrm{d}\tau,\\ &= e^{At_f}\mathbf{x}_0+\int_0^{t_f}e^{A\tau}B\mathbf{u}(t_f-\tau)\mathrm{d}\tau. \end{split}\]

Now, there are many possible definitions of what is meant by the ‘controllability’ of a given system and we shall adopt the following one.

Given \(t_f\) and \(\displaystyle\mathbf{x}_0, \mathbf{x}_f\) which are column \(n\)-vectors, the system is said to be controllable on \(\displaystyle [0,t_f]\) if there is a control \(\displaystyle\mathbf{u}(t)\) defined on \(\displaystyle[0,t_f]\) such that, if \(\displaystyle\mathbf{x}(0)=\mathbf{x}_0\), then \(\displaystyle\mathbf{x}(t_f)=\mathbf{x}_f\). In order that the behaviour of the system is relatively ``smooth’’, we usually impose the condition that \(\displaystyle\mathbf{u}(t)\) is piecewise continuous \(\displaystyle[0, t_f]\) (i.e., continuous except at a finite number of points).

We shall shortly see that the controllability or otherwise of a system of the above form depends on the \(\displaystyle n\times n\) matrix

\[ W(0,t_f) = \int_0^{t_f}e^{A\tau}BB^Te^{A^T\tau}\mathrm{d}\tau. \]

In fact, a system of the given form is completely controllable on \([0, t_f]\) if and only if the matrix \(\displaystyle W(0,t_f)\) is non-singular (i.e., \(\displaystyle W^{-1}(0,t_f)\) exists).

Suppose first of all that \(\displaystyle W(0,t_f)\) is non-singular. Then, given \(\displaystyle t_f, \mathbf{x}_0\) and \(\mathbf{x}_f\), we can form the control

\[ \mathbf{u}(t) = B^Te^{A^T(t_f-t)}W^{-1}(0,t_f)\left[\mathbf{x}_f-e^{At_f}\mathbf{x}_0\right]. \]

Substituting this control into (17) we see that

\[\begin{split} \mathbf{x}_f &= e^{At_f}\mathbf{x}_0 + \int_0^{t_f}e^{A\tau}BB^Te^{A^T\tau}W^{-1}(0,t_f)\left[\mathbf{x}_f-e^{At_f}\mathbf{x}_0\right]\mathrm{d}\tau,\\ &= e^{At_f}\mathbf{x}_0 + \int_0^{t_f}e^{A\tau}BB^Te^{A^T\tau}\mathrm{d}\tau\cdot W^{-1}(0,t_f)\left[\mathbf{x}_f-e^{At_f}\mathbf{x}_0\right],\\ &=e^{At_f}\mathbf{x}_0 + W(0,t_f)W^{-1}(0,t_f)\left[\mathbf{x}_f-e^{At_f}\mathbf{x}_0\right],\\ &=e^{At_f}\mathbf{x}_0+\mathbf{x}_f-e^{At_f}\mathbf{x}_0 = \mathbf{x_f}, \end{split}\]

i.e. the control \(\displaystyle\mathbf{u}(t)\) takes the system from \(\displaystyle\mathbf{x}(0)=\mathbf{x}_0\) to \(\displaystyle\mathbf{x}(t_f)=\mathbf{x}_f\), as required. The system is, therefore, completely controllable on \([0, t_f]\).

On the other hand, suppose that \(\displaystyle W(0,t_f)\) is singular, i.e. \(\displaystyle |W(0,t_f)|=0\). Then, there is a vector \(\displaystyle \mathbf{x}_1\neq 0\) such that

\[ \mathbf{x}_1^TW(0,t_f)=0. \]

Indeed, \(\displaystyle\mathbf{x}_1^TW(0,t_f)\mathbf{x}_1=0\), so that

\[\begin{split} \mathbf{x}_1^T\int_0^{t_f}e^{A\tau}BB^Te^{A^T\tau}\mathrm{d}\tau\mathbf{x}_1 &= \int_0^{t_f}\mathbf{x}_1^Te^{A\tau}BB^Te^{A^T\tau}\mathbf{x}_1\mathrm{d}\tau,\\ &=\int_0^{t_f}(B^Te^{A^T\tau}\mathbf{x}_1)^T(B^Te^{A^T\tau}\mathbf{x}_1)\mathrm{d}\tau,\\ &= \int_0^{t_f}||B^Te^{A^T\tau}\mathbf{x}_1||^2\mathrm{d}\tau = 0. \end{split}\]

This is only possible if \(\displaystyle\left(B^Te^{A^Tt}\mathbf{x}_1\right)^T=\mathbf{x}^T_1e^{At}B=0\) on \(\displaystyle [0, t_f]\).

Now, if the given system were completely controllable on \(\displaystyle [0, t_f]\), we should be able to find a control \(\displaystyle\mathbf{u}(t)\) which would take it from \(\displaystyle\mathbf{x}_0=0\) at \(t=0\) to \(\displaystyle\mathbf{x}=\mathbf{x}_1\) at \(t=t_f\), i.e.

\[ \mathbf{x}_1=e^{At_f}\cdot 0+\int_0^{t_f}e^{A\tau}B\mathbf{u}(t_f-\tau)\mathrm{d}\tau. \]

Then

\[\begin{split} \mathbf{x}_1^T\mathbf{x}_1 &=\mathbf{x}_1^T\int_0^{t_f}e^{A\tau}B\mathbf{u}(t_f-\tau)d\tau\mathbf{x}_1,\\ &= \int_0^{t_f}\mathbf{x}_1^Te^{A\tau}B\mathbf{u}(t_f-\tau)\mathbf{x}_1d\tau = 0. \end{split}\]

This, of course is a contradiction since \(\displaystyle\mathbf{x}_1\neq 0\).

This necessary and sufficient condition for the controllability of the system on \(\displaystyle [0, t_f]\) can be replaced by one which is easier to apply by the observation that \(\displaystyle W(0,t_f)\) is positive definite if and only if there is no non-zero column \(n\)-vector \(\displaystyle\mathbf{x}_1\) such that

\[ \mathbf{x}_1^Te^{At}B=0 \]

for all \(\displaystyle t\in[0, t_f]\).


Example 4.1.1 Consider the system

\[\begin{split} \frac{\mathrm{d}}{\mathrm{d}t}\begin{pmatrix} x_1\\x_2\end{pmatrix} = \begin{pmatrix}1&-1\\1&-1\end{pmatrix}\begin{pmatrix} x_1\\x_2\end{pmatrix}+\begin{pmatrix}1\\0\end{pmatrix} u, \end{split}\]

for which we can show that

\[ A = HJH^{-1} \]

where

\[\begin{split} A=\begin{pmatrix}1&-1\\1&-1\end{pmatrix}, H=H^{-1}=\begin{pmatrix}1&0\\1&-1\end{pmatrix} ~\&~ J=\begin{pmatrix}0&1\\0&0\end{pmatrix}. \end{split}\]

Then

\[\begin{split} e^{At}=He^{Jt}H^{-1}=\begin{pmatrix}1&0\\1&-1\end{pmatrix}\begin{pmatrix}1&t\\0&1\end{pmatrix}\begin{pmatrix}1&0\\1&-1\end{pmatrix}=\begin{pmatrix}1+t&-t\\t&1-t\end{pmatrix}. \end{split}\]

Hence

\[\begin{split} e^{At}B = \begin{pmatrix}1+t&-t\\t&1-t\end{pmatrix}\begin{pmatrix}1\\0\end{pmatrix} = \begin{pmatrix}1+t\\t\end{pmatrix}. \end{split}\]

Suppose now that there is a vector \(\displaystyle\mathbf{x}_1=[\alpha, \beta]^T\) such that

\[\begin{split} \mathbf{x}_1^Te^{At}B = [\alpha, \beta]\begin{pmatrix}1+t\\t\end{pmatrix}=\alpha(1+t)+\beta t = 0. \end{split}\]

From the constant part we see that \(\alpha=0\), while from the coefficient of \(t\) it follows that \(\alpha+\beta=\beta=0\). Since \(\mathbf{x}_1\) cannot be non-zero, we have shown that the above system is controllable.

Let us now consider a particular case in which

\[\begin{split} t_f=1, \mathbf{x}_0=0 \text{ and } \mathbf{x}_f=\begin{pmatrix}3\\1\end{pmatrix}. \end{split}\]

We require

\[\begin{split} W(0,1) &= \int_0^1(e^{A\tau}B)(e^{A\tau}B)^T\mathrm{d}\tau =\int_0^1\begin{pmatrix}1+\tau\\\tau\end{pmatrix}[1+\tau,\tau]\mathrm{d}\tau, \\ &=\int_0^1\begin{pmatrix}(1+\tau)^2&\tau+\tau^2\\\tau+\tau^2&\tau^2\end{pmatrix} \mathrm{d}\tau. \end{split}\]

Here

\[\begin{split} \int_0^1(1+\tau)^2\mathrm{d}\tau &= \left[\frac{(1+\tau)^3}{3}\right]_0^1=\frac{2^3-1^3}{3}=\frac{7}{3},\\ \int_0^1(\tau+\tau^2)\mathrm{d}\tau&=\left[\frac{\tau^2}{2}+\frac{\tau^3}{3}\right]=\frac{1}{2}+\frac{1}{3}=\frac{5}{6}, \end{split}\]

and \(\displaystyle\int_0^1\tau^2\mathrm{d}\tau=\frac{1}{3}\).

Therefore

\[\begin{split} W(0,1)=\begin{pmatrix}\frac{7}{3}&\frac{5}{6}\\\frac{5}{6}&\frac{1}{3}\end{pmatrix}, \end{split}\]

for which \(\displaystyle \left|W(0,1)\right|=\frac{7}{9}-\frac{25}{36}=\frac{1}{12}\) and

\[\begin{split} W^{-1}(0,1) = 12\begin{pmatrix}\frac{1}{3}&-\frac{5}{6}\\-\frac{5}{6}&\frac{7}{3}\end{pmatrix}=\begin{pmatrix}4&-10\\-10&28\end{pmatrix}. \end{split}\]

The control which takes the system from

\[\begin{split} \mathbf{x}_0=0 \text{ to } \mathbf{x}_f=\begin{pmatrix}3\\1\end{pmatrix}, \end{split}\]

is then given by

\[\begin{split} u(t) &= B^Te^{A^T(1-t)}W^{-1}(0,1)\mathbf{x}_f,\\ &= \begin{pmatrix}1+(1-t)& 1-t\end{pmatrix}\begin{pmatrix}4&-10\\-10&28\end{pmatrix}\begin{pmatrix}3\\1\end{pmatrix} =\begin{pmatrix} 2-t& 1-t\end{pmatrix}\begin{pmatrix}2\\-2\end{pmatrix}=2. \end{split}\]

The necessary and sufficient conditions for the controllability of systems of the above form can, in fact, be converted to a much simpler one by remembering that

\[ e^{At}B=\sum_{r=0}^\infty\frac{t^r}{r!}A^rB. \]

By the Cayley-Hamilton Theorem we know that for \(r\geq n\) we have

\[ A^r = \sum_{i=0}^{n-1}\alpha_iA^i, \]

so that \(\displaystyle e^{At}B\) can naturally be determined by using powers of \(A\) up to and including \(\displaystyle A^{n-1}\). Consider now the \(n\times np\) controllability matrix

\[ G = \left[B, AB, A^2B,\ldots,A^{n-1}B\right]. \]

(We’ll soon see why its called this.)

The maximum rank of this matrix is \(n\) and let us suppose that it is actually less than this. Then there is a non-zero column \(n\)-vector \(\mathbf{x}_1\) such that

\[ \mathbf{x}_1^TG = \mathbf{x}_1^T\left[B, AB, A^2B,\ldots, A^{n-1}B\right]=0, \]

i.e.,

\[ \mathbf{x}_1^TB = \mathbf{x}_1^TAB = \mathbf{x}_1^TA^2B=\cdots=\mathbf{x}_1^TA^{n-1}B = 0. \]

Clearly,

\[ \mathbf{x}_1^Te^{At}B=0, \]

and the system under consideration is not controllable.

Alternatively, suppose that a given system is not controllable. Then, there is a non-zero \(n\)-vector \(\mathbf{x}_1\) such that

\[ \mathbf{x}_1^Te^{At}B = \mathbf{x}_1^T\sum_{r=0}^\infty\frac{t^r}{r!}A^rB=0. \]

Clearly, looking at the individual powers of \(t\), \(\displaystyle\mathbf{x}_1^TA^rB=0\) for all \(r\) and, in particular

\[ \mathbf{x}_1^TG = \mathbf{x}_1^T\left[B, AB, A^2B,\ldots,A^{n-1}B\right] = 0, \]

i.e. \(G\) does not have full rank.

We have thus shown that a given system is controllable if and only if the corresponding controllability matrix has full rank.


Example 4.1.2 Suppose that

\[\begin{split} A=\begin{pmatrix}0&1&1\\-1&-2&-1\\3&5&2\end{pmatrix} \text{ and } B=\begin{pmatrix}0\\1\\0\end{pmatrix}. \end{split}\]

Then

\[\begin{split} G = \left[B, AB, A^2B\right] = \begin{pmatrix}0&1&3\\1&-2&-2\\0&5&3\end{pmatrix}, \end{split}\]

and

\[\begin{split} |G|=-1\left|\begin{matrix}1&3\\5&3\end{matrix}\right| = -1(-12) = 12 (\neq 0). \end{split}\]

Since \(G\) is clearly has full rank, the system

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x} +Bu, \]

is controllable.


Example 4.1.3 Let

\[\begin{split} A=\begin{pmatrix}0&1&0\\0&0&1\\0&1&0\end{pmatrix} \text{ and } B=\begin{pmatrix}0\\1\\1\end{pmatrix}. \end{split}\]

Then

\[\begin{split} G = [B, AB, A^2B] = \begin{pmatrix}0&1&1\\1&1&1\\1&1&1\end{pmatrix}, \end{split}\]

which clearly does not have full rank. In this case the corresponding system is not controllable and the reason for this can be seen if we set about diagonalising the system matrix \(A\) by using the fact that

\[ A=HDH^{-1}, \]

where

\[\begin{split} H = \begin{pmatrix}1&1&1\\0&1&-1\\0&1&1\end{pmatrix}, H^{-1}=\frac{1}{2}\begin{pmatrix}2&0&-2\\0&1&1\\0&-1&1\end{pmatrix} ~\&~ D=\begin{pmatrix}0&0&0\\0&1&0\\0&0&-1\end{pmatrix}. \end{split}\]

Then if we take the similarity transformation \(\displaystyle\mathbf{x}'=H^{-1}\mathbf{x}\), we see that the system

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x}+Bu = HDH^{-1}\mathbf{x}+Bu, \]

can be brought to the form

\[ \frac{\mathrm{d}\mathbf{x}'}{\mathrm{d}t}=D\mathbf{x}'+B'u, \]

where

\[\begin{split} B' = H^{-1}B = \frac{1}{2}\begin{pmatrix}2&0&-2\\0&1&1\\0&-1&1\end{pmatrix}\begin{pmatrix}0\\1\\1\end{pmatrix}=\begin{pmatrix}-1\\1\\0\end{pmatrix}. \end{split}\]

We then have

\[\begin{split} \frac{\mathrm{d}}{\mathrm{d} t}\begin{pmatrix} x'_1\\x'_2\\x'_3\end{pmatrix} = \begin{pmatrix}0&0&0\\0&1&0\\0&0&-1\end{pmatrix}\begin{pmatrix} x'_1\\x'_2\\x'_3\end{pmatrix}+\begin{pmatrix}-1\\1\\0\end{pmatrix} u, \end{split}\]

and this shows that the system has a mode, \(x'_3\), which is not affected by the input and is, therefore, uncontrollable.


Controllability of Discrete Systems#

Discussions about controllability (and observability - see below) can also be related to systems of difference equations which take the form

\[\begin{split} \mathbf{x}_{k+1} &=A\mathbf{x}_k+B\mathbf{u}_k,\\ \mathbf{y}_k &= C\mathbf{x}_k. \end{split}\]

The solution of the first part of the system is given by

\[ \mathbf{x}_k = A^k\mathbf{x}_0+\sum_{r=0}^{k-1}A^rB\mathbf{u}_{k-r-1}. \]

Given \(\mathbf{x}_0\) and \(\mathbf{x}_f\) and taking \(t_f=n\), we can determine a sequence of controls \(\displaystyle\mathbf{u}_0, \mathbf{u}_1, \ldots, \mathbf{u}_{n-1}\) which take the system to \(\displaystyle\mathbf{x}_n=\mathbf{x}_f\) if we can solve the system of equations

\[ \mathbf{x}_f=A^n\mathbf{x}_0+\sum_{r=0}^{n-1}A^rB\mathbf{u}_{n-r-1}, \]

i.e.

\[\begin{split} \mathbf{x}_f - A^n\mathbf{x}_0 &= \sum_{r=0}^{n-1}A^rB\mathbf{u}_{n-r-1},\\ &=\begin{pmatrix} B&AB&A^2B&\ldots&A^{n-1}B\end{pmatrix}\begin{pmatrix}\mathbf{u}_{n-1}\\\mathbf{u}_{n-2}\\\vdots\\\mathbf{u}_0\end{pmatrix},\\ &= G\tilde{\mathbf{u}}, \end{split}\]

where

\[\begin{split} \tilde{\mathbf{u}} = \begin{pmatrix} \mathbf{u}_{n-1}\\\mathbf{u}_{n-2}\\\vdots\\\mathbf{u}_0\end{pmatrix}. \end{split}\]

Such a system can be solved for any \(\mathbf{x}_0\) and \(\mathbf{x}_f\) if rank \(G=n\) and this is again seen to be a necessary and sufficient condition for controllability.

Example 4.1.4 Let \(A\) and \(B\) be the matrices in example 4.2. Clearly the corresponding system of difference equations

\[ \mathbf{x}_{k+1} = A\mathbf{x}_k+Bu_k, \]

is controllable and, as an example, let us take \(\displaystyle\mathbf{x}_0=\mathbf{0}\) and \(\displaystyle\mathbf{x}_f=\begin{pmatrix}5&-1&1\end{pmatrix}^T\).

We then have to solve the system of equations

\[\begin{split} \begin{pmatrix}5\\-1\\1\end{pmatrix} = \begin{pmatrix}0&1&3\\1&-2&-2\\0&5&3\end{pmatrix}\begin{pmatrix} u_2\\u_1\\u_0\end{pmatrix}, \end{split}\]

for which the augmented matrix is

\[\begin{split} \begin{pmatrix}0&1&3&|&5\\1&-2&-2&|&-1\\0&5&3&|&1\end{pmatrix} \to \begin{pmatrix}0&1&3&|&5\\1&-2&-2&|&-1\\0&0&-12&|&-24\end{pmatrix} \to \begin{pmatrix}1&-2&-2&|&-1\\0&1&3&|&5\\0&0&1&|&2\end{pmatrix}, \end{split}\]

clearly \(u_0=2\), \(u_1+3u_0=5\implies u_1=-1\) and

\[ u_2-2u_1-2u_0 = -1 \implies u_2=1. \]

We can show that the control sequence \(\displaystyle\{2,-1,1\}\) has the desired effect by substituting into the system equations

\[ \mathbf{x}_{k+1} = A\mathbf{x}_k+Bu_k. \]

Then

\[\begin{split} \mathbf{x}_1 &= A\mathbf{x}_0+Bu_0 = 0+\begin{pmatrix}0\\1\\0\end{pmatrix} 2 = \begin{pmatrix}0\\2\\0\end{pmatrix},\\ \mathbf{x}_2 &= A\mathbf{x}_1+Bu_1 = \begin{pmatrix}0&1&1\\-1&-2&-1\\3&5&2\end{pmatrix}\begin{pmatrix}0\\2\\0\end{pmatrix}+\begin{pmatrix}0\\1\\0\end{pmatrix}(-1)=\begin{pmatrix}2\\-5\\10\end{pmatrix},\\ \mathbf{x}_3 &= A\mathbf{x}_2+Bu_2 = \begin{pmatrix}0&1&1\\-1&-2&-1\\3&5&2\end{pmatrix}\begin{pmatrix}2\\-5\\10\end{pmatrix}+\begin{pmatrix}0\\1\\0\end{pmatrix}\cdot1 =\begin{pmatrix}5\\-1\\1\end{pmatrix}. \end{split}\]

In this case the desired control cannot be achieved until \(k=n=3\) but sometimes it can be achieved easier. This depends on the rank of the ``reduced’’ controllability matrix

\[ G_k = \begin{pmatrix} B&AB&A^2B&\ldots&A^{k-1}B\end{pmatrix}. \]

Observability#

We have so far considered how the state vector \(\mathbf{x}\) of a system is related to the control \(\mathbf{u}\) for both continuous and discrete systems.

Suppose we have a system of differential equations of the form

Note

Our discussion on observability will focus on the continuous case; however, just as with controllability, the same arguments apply to discrete systems of the form \(\displaystyle \mathbf{x}_{k+1} = A\mathbf{x}_k+B\mathbf{u}_k\).

\[ \frac{\mathrm{d}\mathbf{x}}{dt} = A\mathbf{x}+B\mathbf{u}. \]

In some cases not all of the state variables can be measured or observed, but we can measure certain output, or observed variables, which are combinations of them to form an output \(l\)-vector

\[ \mathbf{y}= C\mathbf{x}, \]

where \(C\) is an \(l\times n\) matrix.

We say that the system under consideration is observable on \(\displaystyle [0, t_f]\) if, for any initial state \(\mathbf{x}_0\), knowledge of \(\mathbf{u}(t)\) and \(\mathbf{y}(t)\) for \(\displaystyle t\in[0, t_f]\) enables \(\mathbf{x}_0\) to be determined uniquely.

Discussions of observability follows similar lines to those of controllability. For example, it can be shown that a system is observable on \([0,t_f]\) if and only if the matrix

\[ M(0,t_f) = \int_0^{t_f}e^{A^T\tau}C^TCe^{A\tau}\mathrm{d}\tau, \]

is non-singular.

This condition is equivalent to the requirement that there is no non-zero \(n\)-vector \(\mathbf{x}_1\) such that

\[ \mathbf{x}_1^Te^{A^Tt}C^T = 0 \qquad (\text{ or } Ce^{At}\mathbf{x}_1=0), \]

for all \(\displaystyle t\in[0,t_f]\).

Again, the actual interval turns out to be irrelevant and we can talk simply in terms of the observability of the system. In this case it then follows that a given system is observable if and only if the \(ln\times n\) observability matrix

\[\begin{split} \theta = \begin{pmatrix} C\\CA\\CA^2\\\vdots\\CA^{n-1}\end{pmatrix}, \end{split}\]

has full rank.


Example 4.1.5 If

\[\begin{split} A = \begin{pmatrix}0&1&1\\-1&-2&-1\\3&5&2\end{pmatrix} \text{ and } C=\begin{pmatrix}1&0&0\end{pmatrix}, \end{split}\]

i.e. the observed variable

\[\begin{split} y=C\mathbf{x} = \begin{pmatrix}1&0&0\end{pmatrix}\begin{pmatrix} x_1\\x_2\\x_3\end{pmatrix} = x_1, \end{split}\]

then

\[\begin{split} \theta = \begin{pmatrix} C \\CA\\CA^2\end{pmatrix} = \begin{pmatrix}1&0&0\\0&1&1\\2&3&1\end{pmatrix}. \end{split}\]

Since

\[\begin{split} |\theta|=1\left|\begin{matrix}1&1\\3&1\end{matrix}\right|=-2\neq 0, \end{split}\]

the given system is observable.
