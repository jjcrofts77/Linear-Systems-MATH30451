3.4 Inhomogeneous Systems of ODEs¶

The aim of this section is to produce a standard result for the solution of a linear system of first order differential equations of the form

(1)¶\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d} t} = A\mathbf{x}+\mathbf{f}(t)\quad :\quad \mathbf{x}(0)=\mathbf{x}_0. \]

Here \(\mathbf{x}\) and \(\mathbf{f}(t)\) are \(n\)-vectors and \(A\) is an \(n\times n\) matrix.

Of course, once this result has been obtained, it can be used to solve systems of higher order differential equations since, as we know, such a system can be reduced to one consisting of first order differential equations.

Before considering the general case, it is illustrative to remind ourselves about the solution of a single differential equation of the form

\[ \frac{\mathrm{d} x}{\mathrm{d} t} = ax + f(t) \quad :\quad x(0)=x_0. \]

Rearranging this differential equation in the form

\[ \frac{\mathrm{d} x}{\mathrm{d} t}-ax=f(t), \]

and multiplying by the integrating factor \(\displaystyle e^{-at}\) gives

\[ \frac{\mathrm{d}}{\mathrm{d} t}\left(e^{-at}x\right) = e^{-at}f(t). \]

Then

\[ e^{-at}x = \int_0^t{e^{-a\tau}f(\tau)\mathrm{d}\tau} + x_0. \]

Here we have used the initial condition to determine the constant of integration.

We then see that

\[\begin{split} x(t) &= e^{at}\left\{x_0+\int_0^t{e^{-a\tau}f(\tau)\mathrm{d}\tau}\right\},\\ &= e^{at}x_0 +\int_0^t{e^{a(t-\tau)}f(\tau)\mathrm{d}\tau}. \end{split}\]

Example 3.4.1 The differential equation

\[ \frac{\mathrm{d} x}{\mathrm{d} t}=3x+e^{-2t}\quad :\quad x(0)=2 \]

has solution

\[\begin{split} x(t) &= e^{3t}\left\{2+\int_0^t{e^{-3\tau}e^{-2\tau}\mathrm{d}\tau}\right\},\\ &= e^{3t}\left\{2+\int_0^t{e^{-5\tau}\mathrm{d}\tau}\right\},\\ &= e^{3t}\left\{2+\left[-\frac{e^{-5\tau}}{5}\right]^t_0\right\} = e^{3t}\left\{2+\left(-\frac{e^{-5t}}{5}+\frac{1}{5}\right)\right\},\\ &= 2e^{3t}-\frac{e^{-2t}}{5}+\frac{e^{3t}}{5} = \frac{11}{5}e^{3t}-\frac{e^{-2t}}{5}. \end{split}\]

Let us now turn our attention to the original system (1). Taking Laplace transforms of this system gives

\[ s\bar{\mathbf{x}}-\mathbf{x}_0 = A\bar{\mathbf{x}}+\bar{\mathbf{f}}(s), \]

i.e.

\[ (sI_n-A)\bar{\mathbf{x}} = \mathbf{x}_0+\bar{\mathbf{f}}(s), \]

which gives

\[ \bar{\mathbf{x}} = (sI_n-A)^{-1}\left[\mathbf{x}_0-\bar{\mathbf{f}}(s)\right]. \]

As we saw previously,

\[ L^{-1}\left[(sI_n-A)^{-1}\right] = e^{At}, \]

therefore \(\displaystyle L^{-1}\left[(sI_n-A)^{-1}\mathbf{x}_0\right] = e^{At}\mathbf{x}_0\), as before. Also, using convolution

\[ L^{-1}\left[(sI_n-A)^{-1}\bar{\mathbf{f}}(s)\right] = \int_0^t{e^{A(t-\tau)}\mathbf{f}(\tau)\mathrm{d}\tau}. \]

Putting these two results together gives

\[ \mathbf{x}(t) = e^{At}\mathbf{x}_0+\int_0^t{e^{A(t-\tau)}\mathbf{f}(\tau)\mathrm{d}\tau}. \]

Obviously, to use this formula we have to be able to determine \(\displaystyle e^{At}\) and this was considered previously to give the result that, if

\[ A=HJH^{-1}, \]

then

\[ e^{At} = He^{Jt}H^{-1}. \]

Example 3.4.2 If

\[\begin{split} A=\begin{pmatrix}-2&1&-1\\0&0&-2\\-4&2&0\end{pmatrix} = HJH^{-1}, \end{split}\]

where

\[\begin{split} H=\begin{pmatrix}1&-1&0\\1&-2&0\\1&0&1\end{pmatrix}, H^{-1}=\begin{pmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{pmatrix} ~\&~ J=\begin{pmatrix}-2&0&0\\0&0&1\\0&0&0\end{pmatrix}, \end{split}\]

then

\[\begin{split} e^{At} = \begin{pmatrix}1&-1&0\\1&-2&0\\1&0&1\end{pmatrix} \begin{pmatrix} e^{-2t}&0&0\\0&1&t\\0&0&1\end{pmatrix} \begin{pmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{pmatrix}. \end{split}\]

It is not always necessary to determine \(\displaystyle e^{At}\); sometimes it is best to consider the problem under consideration first.


Example 3.4.3 Suppose that we have a system of the form (1) for which \(A\) is the matrix in the example above, \(\displaystyle\mathbf{f}=[0,1,0]^T\) and \(\displaystyle\mathbf{x}_0=\mathbf{0}\). Then

\[\begin{split} \mathbf{x}(t) &= e^{At}\left\{0+\int_0^t{e^{-A\tau}\mathbf{f}\mathrm{d}\tau}\right\},\\ &=\int_0^t{He^{J(t-\tau)}H^{-1}\mathbf{f}\mathrm{d}\tau} = H\int_0^t{e^{J(t-\tau)}\mathrm{d}\tau}H^{-1}\mathbf{f}. \end{split}\]

Now,

\[\begin{split} H^{-1}\mathbf{f} = \begin{pmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{pmatrix}\begin{pmatrix}0\\1\\0\end{pmatrix} = \begin{pmatrix}-1\\-1\\1\end{pmatrix}. \end{split}\]

Also $\( \int_0^t{e^{J(t-\tau)}\mathrm{d}\tau} = \int_0^t\begin{pmatrix} e^{-2(t-\tau)}&0&0\\0&1&t-\tau\\0&0&1\end{pmatrix} \mathrm{d}\tau =\begin{pmatrix}\frac{1}{2}\left(1-e^{-2t}\right)&0&0\\0&t&\frac{t^2}{2}\\0&0&t\end{pmatrix}. \)$

(In the above, integration is computed component wise.)

Hence,

\[\begin{split} \int_0^t{e^{J(t-\tau)}\mathrm{d}\tau}H^{-1}\mathbf{f} = \begin{pmatrix}\frac{1}{2}\left(1-e^{-2t}\right)&0&0\\0&t&\frac{t^2}{2}\\0&0&t\end{pmatrix} \begin{pmatrix}-1\\-1\\1\end{pmatrix}= \begin{pmatrix}-\frac{1}{2}(1-e^{-2t})\\-t+\frac{t^2}{2}\\t\end{pmatrix} \end{split}\]

and

\[\begin{split} \mathbf{x}(t) &= H\int_0^t{e^{J(t-\tau)}\mathrm{d}\tau}H^{-1}\mathbf{f} = \begin{pmatrix}1&-1&0\\1&-2&0\\1&0&1\end{pmatrix} \begin{pmatrix}-\frac{1}{2}(1-e^{-2t})\\-t+\frac{t^2}{2}\\t\end{pmatrix},\\ &= \begin{pmatrix} t-\frac{t^2}{2}-\frac{1}{2}(1-e^{-2t})\\ 2t-t^2-\frac{1}{2}(1-e^{-2t})\\ t-\frac{1}{2}(1-e^{-2t}) \end{pmatrix} \end{split}\]
