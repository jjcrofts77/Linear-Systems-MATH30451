2.1 Similarity Transformations and the Jordan Canonical Form#

Recall that if \(A\) is a real symmetric matrix, and if we choose \(P\) to be the matrix whose columns are the orthonormal eigenvectors of \(A\), then the matrix \(D=P^{-1}AP\) is diagonal, with non-zero elements given by the eigenvalues of \(A\). However, if the matrix \(A\) is not symmetric then diagonalisation might not be possible. For such systems, a structure, which is close to diagonal form, known as the Jordan canonical form, can be obtained.

An \(n\times n\) matrix is diagonalisable if and only if the sum of the dimensions of the eigenspaces is \(n\). Or, equivalently, if and only if \(A\) has \(n\) linearly independent eigenvectors. As we saw in DEM (DETM) not all matrices are diagonalisable.


Background Material#

Before moving on let us review some key ideas from linear algebra.


Let \(f\) be a linear map from a vector space \(V\) to itself, and \(A\) the matrix representing \(f\) w.r.t the standard basis.

Important

Hopefully most of this is familiar but if not don’t worry! We’ll cover it in the lectures in more detail.

  1. Then the characteristic polynomial is \(\displaystyle\chi_A(t) = \det{(A-tI)}\).

  2. \(\lambda\) is an eigenvalue of \(A\) if \(\displaystyle\chi_A(\lambda)=0\).

  3. \(v\) is an eigenvector for the eigenvalue \(\lambda\) if \(\displaystyle Av=\lambda v\).

  4. The eigenspace for the eigenvalue \(\lambda\) is \(\displaystyle\ker{(A-\lambda I)}\).


Example 2.1.1 The matrices

\[\begin{split} A_1 = \begin{pmatrix} 1~ &0\\0~ & 1\end{pmatrix} \quad\text{and}\quad A_2 = \begin{pmatrix} 1~ &1\\0~ & 1\end{pmatrix} \end{split}\]

have the same characteristic polynomial given by

\[ \chi_{A_1}(t) = \chi_{A_2}(t) = (t-1)^2 \]

which implies that \(\displaystyle \lambda=1\) has AM=2 in both cases.

The corresponding eigenspaces are given by

  • \(A_1:\)

\[\begin{split} (I-A_1)\mathbf{v} = \begin{pmatrix} 0~ & 0\\0~&0\end{pmatrix}\mathbf{v}=\mathbf{0} \end{split}\]

which \(\implies\) \(v_1 = \alpha\) and \(v_2 = \beta\). Therefore

\[\begin{split} \text{Eigenspace - }\quad \left\{ \alpha\begin{pmatrix} 1\\0\end{pmatrix}+\beta\begin{pmatrix} 0\\1\end{pmatrix} : \alpha, \beta\in\mathbb{R} \right\} \end{split}\]

and the GM=2. (\(\therefore A_1\) is diagonalisable (trivially).)

  • \(A_2\):

\[\begin{split} (I-A_2)\mathbf{v} = \begin{pmatrix} 0~ & -1\\0~&0\end{pmatrix}\mathbf{v}=\mathbf{0} \end{split}\]

which \(\implies\) \(v_1 = \alpha\) and \(v_2 =0\). Therefore

\[\begin{split} \text{Eigenspace - }\quad \left\{ \alpha\begin{pmatrix} 1\\0\end{pmatrix} : \alpha\in\mathbb{R} \right\} \end{split}\]

and the GM=1. (NOT diagonalisable.)


Theorem 2.1.1 (Cayley-Hamilton) A matrix satisfies its own characteristic equation, i.e., \(\displaystyle\chi_A(A)=0\).


Example 2.1.2 Let \(\displaystyle A = \begin{pmatrix} 1~&1\\2~&1\end{pmatrix}\) then

\[\begin{split} \chi_A(t) &= \left| \begin{array}{cc}t-1&1\\2&t-1\end{array}\right|\\ &= (t-1)^2-2\\ &= t^2-2t-1 \end{split}\]

Now to check C-H we compute

\[\begin{split} \chi_A(A) &= \begin{pmatrix} 1~&1\\2~&1\end{pmatrix}^2-2\begin{pmatrix}1~&1\\2~&1\end{pmatrix}-\begin{pmatrix}1~&0\\0~&1\end{pmatrix}\\ &= \begin{pmatrix}3~&2\\4~&3\end{pmatrix}-\begin{pmatrix}2~&2\\4~&2\end{pmatrix}-\begin{pmatrix}1~&0\\0~&1\end{pmatrix}\\ &= \begin{pmatrix}0~&0\\0~&0\end{pmatrix} \end{split}\]

as expected.

It is sometimes useful to use C-H to compute matrix powers.


Example 2.1.3 For \(\displaystyle A = \begin{pmatrix}1&2\\3&4\end{pmatrix}\) we have that

\[\begin{split} \chi_A(t) = \left|\begin{array}{cc}1-t&-2\\-3&4-t\end{array}\right| = t^2-5t-2 \end{split}\]

so that

(13)#\[ A^2-5A-2I_2 = 0\implies A^2 = 5A+2I_2. \]

Note

We denote by \(I_n\) the \(n\times n\) identity matrix

Now using (13) we can work out any power, \(\displaystyle A^k\), in terms of just \(A\) and \(I_2\).

So, for example,

\[\begin{split} A^3 &= (5A+2I_2)A = 5A^2+2A = 5(5A+2I_2)+2A = 27A+10I_2\\ A^4 &= A^3A = (27A+10I_2)A = 27A^2+10A = \cdots = 145A+54I_2. \end{split}\]

etc.

We can also work out the inverse of \(A\):

\[ A^2=5A+2I_2\quad\implies\quad A^{-1} = \frac{A-5I_2}{2}. \]

We shall make good use of the Caley-Hamilton theorem throughout this course!


Another concept that will prove useful is that of the so-called minimal polynomial which we define below.


Monic polynomials

  • \(\displaystyle 3t^2+t-1\) is not monic

  • \(\displaystyle t^2+t-2\) is monic

Definition 2.1.2 The minimal polynomial, \(\displaystyle m_A(t)\), of \(\displaystyle A\) is the monic polynomial of least degree for which \(\displaystyle m_A(A)=0\).


Note that monic means that the coefficient of the term with highest power in \(\displaystyle m_A(t)\) equals one.

Note

  • The minimal polynomial of \(A\) is unique, and divides the characteristic polynomial.

  • This means that all the factors of \(\displaystyle m_A(t)\) are also factors of \(\displaystyle \chi_A(t)\).

  • We can use this to determine \(\displaystyle m_A(t)\).

  • If \(P\) is any invertible matrix, then \(\displaystyle m_A(P^{-1}AP)=0\).


Theorem 2.1.3 The minimal polynomial has the form

\[m_A(t) = (t-\lambda_1)^{e_1}\cdots(t-\lambda_{n_k})^{e_k} \]

where \(\lambda_1,\ldots, \lambda_{n_k}\) are the eigenvalues of the matrix \(A\), and the numbers \(e_i\) are such that \(1\leq e_i\leq r_i\) with \(r_i\) the algebraic multiplicity of \(\lambda_i\). Moreover, \(A\) is diagonalisable if and only if each \(e_i=1\).


Example 2.1.4 Consider \(\displaystyle \chi_{A_2}(t) = (t-1)^2\) then from the above we know that \(\displaystyle m_{A_2}(t)\) is either

\[ \text{(i) } (t-1)^2 \quad\text{or} \quad \text{(ii) } t-1. \]

Let’s check

\[\begin{split} A_2 - I_2 = \begin{pmatrix} 1~& 1\\0~& 1\end{pmatrix} - \begin{pmatrix} 1~& 0\\0~& 1\end{pmatrix} = \begin{pmatrix} 0~& 1\\0~& 0\end{pmatrix}\neq 0 \end{split}\]

Therefore \(\displaystyle m_{A_2}(t) = (t-1)^2\).


Example 2.1.5 Now suppose we have \(\displaystyle A_3 = \begin{pmatrix} 1~&1~&0\\0~&1~&0\\0~&0~&1\end{pmatrix}.\)

Then

\[\begin{split} \chi_{A_3}(t) = \left|\begin{array}{ccc}t-1&-1&0\\0&t-1&0\\0&0&t-1\end{array}\right| = (t-1)^3\end{split}\]

and so \(m_{A_3}(t)\) can be one of

\[ \text{(i) } (t-1)^3, \quad \text{(ii) } (t-1)^2 \quad \text{or} \quad \text{(iii) } t-1. \]

First we check

\[\begin{split} (A_3-I_3)^2 = \begin{pmatrix} 0~&1~&0\\0~&0~&0\\0~&0~&0 \end{pmatrix}^2 = \begin{pmatrix} 0~&0~&0\\0~&0~&0\\0~&0~&0 \end{pmatrix}. \end{split}\]

Therefore \(\displaystyle m_{A_3}(t) = (t-1)^2 = t^2-2t+1\).


Importantly, if the factor \((t-\lambda_i)\) appears in the minimal polynomial with multiplicity \(e_i\) (i.e., \(\displaystyle m_A(t)=(t-\lambda_i)^{e_i}g(t)\) for some polynomial \(g(t)\) which does not have \((t-\lambda_i)\) as a factor), then the generalised eigenspace is \(\displaystyle\ker\left[(A-\lambda_i I)^{e_i}\right]\).


Proposition 2.1.4 Using the notation from Theorem 2.1.3

\[ \ker(A-\lambda_iI)\subset\ker\left[(A-\lambda_iI)^2\right]\subset\cdots \subset\ker\left[(A-\lambda_iI)^{e_i}\right]. \]

Theorem 2.1.5 (Primary Decomposition Theorem) Let \(\displaystyle f:V\mapsto V\) be linear and represented by the matrix \(A\) with minimal polynomial \(\displaystyle m_A(t) = (t-\lambda_1)^{e_1}\ldots(t-\lambda_r)^{e_r}\) where \(\displaystyle \lambda_1,\ldots, \lambda_r\) are distinct eigenvalues of \(A\) and \(\displaystyle e_1,\ldots,e_r\) are natural numbers. Let \(\displaystyle V_1,\ldots, V_r\) denote the generalised eigenspace (\(\displaystyle V_i=\ker\left[(A-\lambda_i)^{e_i}\right]\)). Then \(\displaystyle V = V_1\bigoplus V_2 \cdots\bigoplus V_r\).


Note

The above theorem says that two (or more) eigenvectors with distinct eigenvalues are linearly independent (among other things).


Example 2.1.6 Suppose \(\displaystyle A = \begin{pmatrix}1&1&0\\0&1&0\\0&0&2\end{pmatrix}\) then it is straightforward to show (exercise) that

\[ \chi_A(t) = m_A(t) = (t-1)^2(t-2). \]

Let us compute the relevant eigenspaces.

\(\lambda_2=2:\) We need to solve

\[\begin{split} (A-2I_3)\mathbf{u} = \begin{pmatrix}-1&1&0\\0&-1&0\\0&0&0\end{pmatrix}\mathbf{u} = \mathbf{0}. \end{split}\]

Clearly, \(u_2 = u_1 = 0\) and so

\[\begin{split} \mathrm{Ker}\left[(A-2I_3)\right] = \Bigg\{\alpha\begin{pmatrix}0\\0\\1\end{pmatrix}: \alpha\in\mathbb{R}\Bigg\}. \end{split}\]

\(\lambda_1=1:\) We need to solve

\[\begin{split} (A-I_3)\mathbf{u} = \begin{pmatrix}0&1&0\\0&0&0\\0&0&1\end{pmatrix}\mathbf{u} = \mathbf{0}. \end{split}\]

Clearly, \(u_2 = u_3 = 0\) and so

\[\begin{split} \mathrm{Ker}\left[(A-I_3)\right] = \Bigg\{\alpha\begin{pmatrix}1\\0\\0\end{pmatrix}: \alpha\in\mathbb{R}\Bigg\}. \end{split}\]

Since \(\lambda_1=1\) appears more than once in \(m_A(t)\) we need to also compute the generalised eigenspace by solving

\[\begin{split} (A-I_3)^2\mathbf{u} = \begin{pmatrix}0&0&0\\0&0&0\\0&0&1\end{pmatrix}\mathbf{u} = \mathbf{0}. \end{split}\]

Clearly, \(u_3 = 0\) and so

\[\begin{split} \mathrm{Ker}\left[(A-I_3)^2\right] = \Bigg\{\alpha\begin{pmatrix}1\\0\\0\end{pmatrix}+\beta\begin{pmatrix}0\\1\\0\end{pmatrix}: \alpha, \beta \in\mathbb{R}\Bigg\}. \end{split}\]

Using the notation of Theorem 2.1.5 we have that

\[\begin{split} V_1 = \Bigg\{\alpha\begin{pmatrix}1\\0\\0\end{pmatrix}+\beta\begin{pmatrix}0\\1\\0\end{pmatrix}: \alpha, \beta \in\mathbb{R}\Bigg\} \end{split}\]

and

\[\begin{split} V_2 = \Bigg\{\alpha\begin{pmatrix}0\\0\\1\end{pmatrix}: \alpha\in\mathbb{R}\Bigg\}. \end{split}\]

Geometrically, \(V_1\) is the \(x\)-\(y\) plane whilst \(V_2\) is the \(z\)-axis. \(V\) is of course \(\mathbb{R}^3\) and it should be clear that any vector \(\mathbf{v}\in V=\mathbf{R}^3\) can be written as the direct sum:

\[ \mathbf{v} = \mathbf{v}_1+\mathbf{v}_2 \text{ where } \mathbf{v}_1\in V_1, \mathbf{v}_2\in V_2. \]

Also, since \(\displaystyle \mathrm{Ker}\left[A-I_3\right]\) is given by the \(x\)-axis we see that \(\displaystyle \ker[(A-I_3)]\subset\ker[(A-I_3)^2]\) as expected.

The results above will hopefully help to motivate some of the arguments that follow in the next few sections.
